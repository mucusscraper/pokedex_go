# Description

This REPL Pokedex was made following the course on boot.dev for improving knowledge of GO and HTTP requests.

## How it works

Type ```
bash
go run .
```

## Commands

```map``` lists all the 20 next map location areas
```bmap``` lists all the 20 previous map location areas
```explore {location_area}``` lists all the pokemons in a location areas
```catch {pokemon}``` tries to catch a pokemon based on its base experience
```inspect {pokemon}``` gives information about a pokemon caught
```pokedex``` lists the pokemon caught
```help``` describes the available commands
```exit``` exits the program

Go catch'em all!
